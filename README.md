  # Scanner App

Scanner  App Project with different themes
# features used in the app:
- Ui,Ux.
- design patern.
- Null Safety.
- Shared Preference.
- barcode Widget
- Light Mode.
- Dark Mode.
- page Transition.
- flutter Barcode Scanner.
